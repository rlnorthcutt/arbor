## Installation

Download the binary for your platform from the releases page.

## Quick Start

Initialize a new project:

```bash
arbor init --blueprint docs
arbor preview
```

Your docs site will be available at http://localhost:8080.

## Creating Pages

Add Markdown files to `content/docs/` and they will be rendered automatically.
