Arbor is a fast, convention-driven static site generator. Here are the basics.

## Creating content

Run `arbor new blog my-post` to create a new blog post at `content/blog/my-post.md`.

## Building your site

Run `arbor build` to generate static HTML to `public/`.

## Previewing locally

Run `arbor preview` to build and serve your site locally with live reload.
